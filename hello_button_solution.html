Решение задачи по JavaScript

Условие задачи:

Вывести кнопку с текстом "Привет", при нажатии на неё выводим alert с текстом "Привет Мир!"

Решение задачи

Нажмите на кнопку ниже, чтобы увидеть результат:

HTML код кнопки:
<button class="btn btn-success hello-button" id="helloButton">
  <i class="bi bi-hand-wave"></i> Привет
</button>
JavaScript код решения:
// Получаем элемент кнопки по ID
const helloButton = document.getElementById('helloButton');

// Добавляем обработчик события клика
helloButton.addEventListener('click', function() {
  // Выводим alert с сообщением
  alert('Привет Мир!');
});
Объяснение кода:
  1. document.getElementById('helloButton') - находим кнопку по её ID
  2. addEventListener('click', function() {...}) - добавляем обработчик события клика
  3. alert('Привет Мир!') - выводим сообщение во всплывающем окне

Дополнительные варианты решения

Решение с использованием onclick в HTML:
<button onclick="alert('Привет Мир!')">
  Привет
</button>
Решение с использованием jQuery:
$(document).ready(function() {
  $('#helloButton').click(function() {
    alert('Привет Мир!');
  });
});
