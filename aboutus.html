Веб-приложение для вычисления квадрата разности двух чисел

О нас

Наша история

  • 2020: Начало разработки
  • 2021: Первый релиз
  • 2022: Добавление Bootstrap
  • 2023: Адаптивный дизайн
  • 2024: Текущая версия

Лабораторная работа началась как учебный проект в 2020 году. Изначально это было простое консольное приложение на PHP для вычисления квадрата разности двух чисел.

В 2021 году мы добавили веб-интерфейс, а в 2022 году полностью переработали проект с использованием Bootstrap 4 для создания адаптивного дизайна.

Сегодня это полноценное веб-приложение, которое демонстрирует все этапы разработки: от настройки сервера до развертывания на GitHub.

Технологии проекта

  • ✓ Bootstrap 4
  • ✓ HTML5

Демонстрация адаптивных утилит Bootstrap

Как работают d-none и d-sm-block:
Всегда видимый блок:

Этот текст виден на всех размерах экранов.

Скрыт на очень маленьких экранах:

Этот текст скрыт на экранах меньше 576px (класс .d-none .d-sm-block).

На очень маленьких экранах вы не увидите этот блок.

Только для очень маленьких экранов:

Этот текст виден только на экранах меньше 576px (класс .d-block .d-sm-none).

На больших экранах этот блок скрыт.

Поведение на разных экранах:
Класс XS (<576px) SM (≥576px)
.d-none .d-sm-block Скрыто Видно
.d-block .d-sm-none Видно Скрыто
.d-none Скрыто Скрыто
.d-block Видно Видно

Это демонстрирует, как работают адаптивные утилиты Bootstrap для контроля отображения контента на разных устройствах.
